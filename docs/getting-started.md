# 快速开始

BalanceHub 是本地桌面应用。配置和中转站凭据保存在当前电脑的应用数据目录中，换电脑使用时可以通过导入导出迁移。

## 1. 安装应用

打开 [GitHub Releases](https://github.com/NotoChen/BalanceHub/releases/latest)，下载当前系统对应的安装包。

- macOS：下载 `.dmg`。
- Windows：下载 `setup.exe`。
- Linux：按发行版选择 AppImage、deb 或 rpm。

Release 里的 `.sig` 文件是应用内自动更新使用的签名文件，不是手动安装入口。

## 2. 添加第一个中转站

首次启动后，按初始化引导添加中转站。

基础配置顺序是：

1. 中转站类型：当前选择 NewAPI。
2. 中转站地址：填写中转站后台地址。
3. 认证方式：优先 Cookie，其次访问令牌，最后才是 API Key。

填写认证信息后，测试连接会尝试读取站点信息、账号信息和额度信息。连接成功后保存，中转站卡片会展示最新额度。

## 3. 日常使用

- 点击刷新，更新所有中转站额度和账号状态。
- 点击签到，对支持签到的中转站执行签到。
- 右键中转站卡片，可以查看 API Key、用量趋势、请求日志、签到记录和测活记录。
- 在设置里配置代理、通知、自动刷新、自动签到和测活参数。

## 4. 迁移配置

设置里提供导入和导出。导出的配置包含中转站信息和本地应用设置，适合在自己的电脑之间迁移。
